# lotus wallet

```
lotus wallet - Manage wallet 管理你的钱包，lotus wallet是与钱包相关的命令
```

## 使用格式

```
用法：lotus wallet command [command options] [arguments...]
```

## 可用命令

```
   new          生成给定类型的新密钥
   
   list         列出钱包地址
   
   balance      获取账户余额
   
   export       导入密钥
   
   import       包含密钥
   
   default      获取默认钱包地址
   
   set-default  设置默认钱包地址
   
   sign         对一个消息签名
   
   verify       验证消息的签名
   
   delete       从钱包中删除一个账户
   
   market       与市场余额相互作用
   
   help, h      显示命令列表或一个命令的帮助
```
