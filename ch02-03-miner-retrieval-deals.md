# lotus-miner retrieval-deals

lotus-miner retrieval-deals - 管理检索交易和相关配置

## 使用格式

lotus-miner retrieval-deals command [command options] [arguments...]

## 可用命令

```
   selection  配置检索交易建议的接受标准
   
   list       列出此矿工的所有活动检索交易
   
   set-ask    配置提供者的检索请求
   
   get-ask    获取提供者的当前检索请求
   
   help, h    显示命令列表或一个命令的帮助
```
